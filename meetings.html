I taught an easy way to get the binary version of a decimal number:

  1. Integer divide the number
  2. Record the remainder
  3. Repeat on the answer from step 1 until you end up with 0
  4. Write the remainders you got from right to left, and you have the answer!

Example: 43/2 = 21 R1, 21/2 = 10 R1, 10/2 = 5 R0, 5/2 = 2 R1, 2/2 = 1 R0, 1/2 = 0 R1

Answer: 101011

I expanded the process to work for any base of numbers up to hexadecimal, the code is on the drive
